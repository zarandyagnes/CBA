## 1. Egyszerűsített CBA / Simplified CBA

### Alkalmazás
- Adatgyűjtés aránytalanul nagy terhet jelent
- Kisebb projektek
- Előszűrés elegendő

### Egyszerűsítések
- Durva, indikatív becslés
- Előzetes keresletbecslés
- Pénzügyi költségek árnyékárak helyett
- Benefit transfer lehetséges

---

## 2. Költség-hatékonyság elemzés (CEA)

### Definíció
> "A hatások csak naturáliában kerülnek meghatározásra"

### Alkalmazás
- Azonos szakpolitikai cél
- Azonos típusú output
- EU kötelezettség teljesítése

### Mutatók

**Költségminimalizálás:**
```
Mutató = (Beruházási NPV + Működési NPV) / Eredmény naturáliában
```
Legalacsonyabb érték a legjobb.

**Hatásmaximalizálás:**
```
Mutató = Eredmény naturáliában / (Beruházási NPV + Működési NPV)
```
Legmagasabb érték a legjobb.

### Éves beruházási költség (ACC)

```
ACC = Beruházási költség × CRF
CRF = r / [1 - (1+r)^(-n)]
```
Ahol: r = 3%, n = élettartam

### Kiegyenlített költség
```
Arány = Σ(CFi) / Σ(Qi)
```

## 3. Legkisebb költség elemzés (LCA)

> "amennyiben a változatok/projektek ugyanarra az eredményre vezetnek a leghatékonyabb választás egyszerűen a legkisebb költségű változat kiválasztását jelenti"

---

## 4. Többszempontú értékelés (MCA)

### Definíció
> "több, számszakilag össze nem adható kritérium (pl. költségek, intézményi megfelelőség, környezetvédelmi megfontolások, esélyegyenlőség stb.) együttes értékelésével"

### Alkalmazás
- Projektszinten: stratégiai változatok összehasonlítása
- Programszinten: projektek rangsorolása
- Kisebb, összetett projekteknél: önálló eszköz

### Módszertan
1. Szempontok azonosítása és indoklása
2. Súlyok meghatározása (szakértői becslés)
3. Pontskála definiálása
4. Értékelés és aggregálás

### Hatástípusok példák

**Használói:** elért szám, színvonal, komfort  
**Externális:** légszennyezés, klíma, természetvédelem  
**Gazdasági:** költségek, önerő, bevételtermelés  
**Intézményi:** engedélyek, szakemberek, társadalmi elfogadás  
**Fenntarthatósági:** finanszírozás, klímaállóság

### Aggregálás
```
Összesített pontszám = Σ(pontszám × súly)
```

---

## 5. Fajlagos költségkorlát / Unit Cost Limit

> "a projekt várható eredményének egységére meghatározott maximális költségét jelenti"

**Alkalmazás:** Kisebb, homogén projektek  
**Ha teljesül:** További értékelés nem szükséges

## Módszerek összehasonlítása / Comparison

| Módszer | Költségek | Hatások | Externáliák | Alkalmazás |
|---------|-----------|---------|-------------|------------|
| Teljes CBA | Árnyékár | Monetizált | Igen | Nagyméretű, komplex |
| Egyszerűsített CBA | Piaci ár | Részben monetizált | Opcionális | Kisebb |
| CEA/LCA | Piaci ár | Naturália | Kulcsfontosságú | Azonos cél |
| MCA | Pontszám | Pontszám | Pontszám | Összetett |
| Fajlagos korlát | Összeg | Egységnyi | Nem | Homogén, kis |
