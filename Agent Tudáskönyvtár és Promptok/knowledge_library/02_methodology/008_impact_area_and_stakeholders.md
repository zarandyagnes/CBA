# Hatásterület és érintettek azonosítása / Impact Area and Stakeholder Identification

## Forrás / Source

**Dokumentum:** TOP PLUSZ Közgazdasági és Pénzügyi Útmutató  
**Fejezet:** 2.1.4 Hatásterület lehatárolása, 2.1.8 Az érintett szereplők azonosítása  
**Oldal:** 12, 14-15

## Áttekintés / Overview

A hatásterület lehatárolása és az érintett szereplők azonosítása kritikus lépések a költség-haszon elemzésben. Ezek biztosítják, hogy az elemzés minden releváns hatást és érintettet figyelembe vegyen, és támogatják a projekt sikeres megvalósítását.

---

## 1. Hatásterület lehatárolása / Impact Area Delineation

### Forrás / Source
**Fejezet:** 2.1.4 Hatásterület lehatárolása  
**Oldal:** 12

### Definíció / Definition

> "A projekt térbeli hatásterülete az a hely vagy terület, ahol a projektet végrehajtják, illetve ahol ebből eredően a hatását kifejti."

### Követelmény / Requirement

> "A hatásterület lehatárolásánál figyelemmel kell lenni az alkalmazandó módszer követelményeire."

### Hatásterület típusok / Types of Impact Areas

#### 1. Fizikai hatásterület / Physical Impact Area
- A projekt fizikai megvalósításának helyszíne
- Infrastruktúra elhelyezkedése
- Építési terület

#### 2. Közvetlen hatásterület / Direct Impact Area
- Ahol a projekt szolgáltatásai közvetlenül elérhetők
- Közvetlen felhasználók/kedvezményezettek területe
- Szolgáltatási terület

#### 3. Közvetett hatásterület / Indirect Impact Area
- Ahol a projekt közvetett hatásai érvényesülnek
- Gazdasági tovagyűrűző hatások területe
- Környezeti hatások területe

### Lehatárolási szempontok / Delineation Criteria

**Módszertani követelmények:**
- CBA módszer: gazdasági hatások területe
- CEA módszer: költséghatékonyság értékelésének területe
- Környezeti hatásvizsgálat: környezeti hatások területe

**Adminisztratív határok:**
- Települési határok
- Járási/megyei határok
- NUTS 2/3 régiók
- Funkcionális területek

**Funkcionális szempontok:**
- Közlekedési vonzáskörzet
- Szolgáltatási körzet
- Gazdasági kapcsolatok területe
- Környezeti hatások elterjedése

## 2. Érintett szereplők azonosítása / Stakeholder Identification

### Forrás / Source
**Fejezet:** 2.1.8 Az érintett szereplők azonosítása  
**Oldal:** 14-15

### Vademecum ajánlás / Vademecum Recommendation

> "A Vademecum felhívja a figyelmet arra, hogy a projekt által érintett szereplők bevonása a döntéshozatali folyamatba elősegíti a projekt sikeres megvalósítását."

### Előnyök / Benefits

**1. Megbízhatóság és legitimáció:**
> "Az eljárás további előnye, hogy növeli a döntésben résztvevő közigazgatási szervezetek megbízhatóságát és legitimációját"

**2. Társadalmi felelősségvállalás:**
> "valamint erősíti a helyi lakosság projekt iránti társadalmi felelősségvállalását és a társadalmi egyenlőséget"

### Alkalmazás a CBA-ban / Application in CBA

> "Az érintettek azonosítása és a projekt elosztási hatásainak vizsgálata hasznosan egészíti ki a költség-haszon elemzést."

### Gyakorlati megvalósítás / Practical Implementation

> "A gyakorlatban ez egy olyan mátrixot jelenthet, amely összekapcsolja a projekt hatásait az érintett ágazatokkal és az érdekelt szereplőkkel."

---

## Érintett szereplők kategóriái / Stakeholder Categories

### 1. Közvetlen kedvezményezettek / Direct Beneficiaries
- Projekt szolgáltatásainak közvetlen felhasználói
- Helyi lakosság (érintett területen)
- Vállalkozások (érintett területen)

### 2. Közvetett kedvezményezettek / Indirect Beneficiaries
- Gazdasági tovagyűrűző hatások kedvezményezettjei
- Környezeti javulás kedvezményezettjei
- Társadalmi hatások kedvezményezettjei

### 3. Projekt megvalósítók / Project Implementers
- Projekt kedvezményezettje (pályázó)
- Kivitelezők
- Szolgáltatók
- Tanácsadók

### 4. Közigazgatási szereplők / Public Administration
- Központi kormányzat
- Helyi önkormányzatok
- Szakmai hatóságok
- Szabályozó szervek

### 5. Érdekképviseletek / Interest Groups
- Civil szervezetek
- Szakmai szövetségek
- Környezetvédelmi szervezetek
- Fogyasztóvédelmi szervezetek

### 6. Negatívan érintettek / Negatively Affected
- Környezeti károk elszenvedői
- Kiszorított tevékenységek szereplői
- Versenyhátrányba kerülők

## Érintettek-hatások mátrix / Stakeholder-Impact Matrix

### Mátrix struktúra / Matrix Structure

| Érintett csoport | Hatás 1 | Hatás 2 | Hatás 3 | ... | Összesített hatás |
|------------------|---------|---------|---------|-----|-------------------|
| Helyi lakosság | + | + | 0 | ... | ++ |
| Vállalkozások | ++ | - | + | ... | + |
| Önkormányzat | + | + | + | ... | ++ |
| Környezetvédők | + | + | 0 | ... | + |
| ... | ... | ... | ... | ... | ... |

**Jelölések:**
- `++` : Jelentős pozitív hatás
- `+` : Pozitív hatás
- `0` : Semleges/nincs hatás
- `-` : Negatív hatás
- `--` : Jelentős negatív hatás

### Példa: Közösségi közlekedési projekt / Example: Public Transport Project

| Érintett csoport | Utazási idő csökkenés | Környezeti javulás | Gazdasági élénkülés | Építési zaj | Összesített |
|------------------|----------------------|-------------------|-------------------|-------------|-------------|
| Helyi lakosság | ++ | + | + | - | ++ |
| Ingázók | ++ | + | 0 | 0 | ++ |
| Helyi vállalkozások | + | 0 | ++ | - | ++ |
| Önkormányzat | + | + | ++ | 0 | ++ |
| Környezetvédők | 0 | ++ | 0 | - | + |
| Autósok | - | 0 | 0 | 0 | - |

## Elosztási hatások vizsgálata / Distributional Impact Analysis

### Cél / Objective

Megérteni, hogy a projekt hatásai hogyan oszlanak meg a különböző érintett csoportok között.

### Vizsgálati szempontok / Analysis Aspects

**1. Térbeli elosztás / Spatial Distribution**
- Városi vs. vidéki területek
- Fejlett vs. elmaradott régiók
- Központi vs. perifériális területek

**2. Társadalmi elosztás / Social Distribution**
- Jövedelmi csoportok
- Életkori csoportok
- Nemek
- Hátrányos helyzetű csoportok

**3. Ágazati elosztás / Sectoral Distribution**
- Különböző gazdasági ágazatok
- Közszféra vs. magánszféra
- KKV-k vs. nagyvállalatok

**4. Időbeli elosztás / Temporal Distribution**
- Rövid távú vs. hosszú távú hatások
- Generációk közötti elosztás
- Átmeneti vs. tartós hatások

## Gyakorlati alkalmazás / Practical Application

### 1. Hatásterület lehatárolása / Delineating Impact Area

**Lépések:**
1. Azonosítsa a projekt fizikai helyszínét
2. Határozza meg a közvetlen szolgáltatási területet
3. Becsülje meg a közvetett hatások területét
4. Vegye figyelembe az alkalmazandó módszer követelményeit
5. Dokumentálja a lehatárolás indokait

**Eszközök:**
- Térképek
- GIS elemzés
- Közlekedési modellezés
- Gazdasági kapcsolatok elemzése

### 2. Érintettek azonosítása / Identifying Stakeholders

**Lépések:**
1. Listázza ki az összes potenciális érintettet
2. Kategorizálja őket (lásd fent)
3. Értékelje az érintettség mértékét
4. Prioritizálja a kulcsfontosságú érintetteket
5. Tervezze meg a bevonási stratégiát

**Eszközök:**
- Érintett-elemzési mátrix
- Hatás-érintett mátrix
- Érdekeltségi térkép
- Befolyás-érdekeltség mátrix

### 3. Elosztási hatások vizsgálata / Analyzing Distributional Effects

**Lépések:**
1. Azonosítsa a főbb hatásokat
2. Határozza meg az érintett csoportokat
3. Becsülje meg a hatások nagyságát csoportonként
4. Készítse el a hatás-érintett mátrixot
5. Értékelje az elosztási méltányosságot

## Gyakorlati tanácsok / Practical Advice

### Hatásterület lehatárolása:

1. **Kezdje a fizikai helyszínnel:** Ez a legkönnyebben meghatározható
2. **Használjon térképeket:** Vizualizálja a hatásterületet

3. **Vonja be őket korán:** Minél korábban, annál jobb
4. **Legyen transzparens:** Ossza meg az információkat nyíltan

1. **Használjon mátrixot:** Strukturált megközelítés
2. **Legyen objektív:** Pozitív és negatív hatások egyaránt
3. **Dokumentálja:** Minden feltételezést és becslést
4. **Kommunikálja:** Ossza meg az eredményeket az érintettekkel
