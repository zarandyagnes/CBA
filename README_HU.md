# TOP PLUSZ Tudásbázis - Magyar Összefoglaló

## Mi ez a projekt?

Ez a projekt a **TOP PLUSZ Közgazdasági és Pénzügyi Útmutató** (97 oldalas PDF dokumentum) teljes körű feldolgozása és átalakítása egy **strukturált, AI-ágensek által használható tudásbázissá**.

### Projekt célja

A cél egy olyan tudáskönyvtár létrehozása volt, amely:

1. **Megőrzi a forrás pontosságát** - Minden információ visszavezethető az eredeti dokumentumra
2. **Emberek számára érthető** - Tiszta, strukturált markdown formátum
3. **AI ágensek számára használható** - Moduláris, lekérdezhető tudáselemek
4. **Újrafelhasználható** - Minden egység önállóan értelmezhető
5. **Nyomon követhető** - Teljes körű forrás-hivatkozások
